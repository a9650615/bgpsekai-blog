據我所知，利民的散熱器、散熱高在同領域界也算是有相當程度的品質了，加上在 2020 年底買的 AMD 3600 處理器原裝散熱器 ( 逐漸 Intel 化的散熱器 ) 根本完全壓不住 3600 所發出的熱量，使用率只要一高，飆到 90 多度甚至是高到當機都有可能，曾經還想過是不是我的處理器有問題想拿去送修。

直到上個月 618 被京東砸到，一個 FS-140 只要 209 RMB ( 約莫 900 初 ) 還有其他優惠券可以用，比二手還便宜，只好買爆，也就是這次要開箱的 Frost Spirit 140 ( 霜靈 )

![](../../assets/external/i.imgur.com/b99f658310b4.png)

從訂購到東西來大概花了兩周，不過重點是便宜，兩周小 case 拉~

## 開箱環節

箱子的體積不小，本次收到的版本為 20 周年版，不過由於沒有買黑化或者白化版本，其實看不出什麼差別  
![](../../assets/external/imgur.com/1a75ff0b1c92.jpg)

生產日期為 5 月，看來不是庫存呢~  
![](../../assets/external/imgur.com/644fe4d43c71.jpg)

接著打開盒子，映入眼簾的是碩大的散熱排  
![](../../assets/external/imgur.com/d3791bdce035.jpg)

密密麻麻的散熱排，加上四根導熱管，心理散熱能力以經點滿了 ( 密集恐懼症請注意 )

![](../../assets/external/i.imgur.com/1551bfa655bd.png)

接下來看一下底部，撕條記得要撕掉哦  
![](../../assets/external/i.imgur.com/ed5b75f0c96c.png)  
![](../../assets/external/i.imgur.com/2bf0aa5b4538.png)

## 安裝

接下來就準備安裝囉，首先打開配件盒  
![](../../assets/external/i.imgur.com/045790c51a99.jpg)

裡面附有 Intel / AMD 的安扣具，一大堆螺絲 ( 支撐用 )，散熱膏 ( 雖然看起來很普通，但導熱效果不錯 )

說明書  
![](../../assets/external/i.imgur.com/36b8ca527f40.jpg)

再來先拆下 AMD 爛爛的原裝散熱器，幸好還沒變成 AMD 原裝水泥膏，輕鬆拆  
![](../../assets/external/i.imgur.com/6d87e7eea681.jpg)  
![](../../assets/external/i.imgur.com/ca70e394f084.jpg)

接下來套上塑膠套管及鎖上螺絲，這段有點高難度，需要雙手並用，不然會一直掉  
![](../../assets/external/i.imgur.com/7e61eec061e7.jpg)

鎖上扣具  
![](../../assets/external/i.imgur.com/47b31e58afd2.jpg)

鎖上散熱器的上下螺絲  
![](../../assets/external/i.imgur.com/51c6d553f5eb.jpg)

接下來準備裝上風扇，分別為 14cm 和 12cm  
![](../../assets/external/i.imgur.com/615813538459.jpg)

將鐵絲扣環扣上風扇並且綁在散熱片的凹槽內，需要施點力量才裝得上去，加上我的殼又很小，我是伸了螺絲起子勾著才裝成功的  
![](../../assets/external/i.imgur.com/e58948b4e9a2.jpg)

最後附上完成圖，登登  
![](../../assets/external/i.imgur.com/a316e57da19e.png)

結果高度還比我的殼還高，只能不裝殼了，接下來看一下換完成的成果。

## 成果

首先是原裝散熱器的溫度，室內 27 度，光待機就 60 度起跳  
![](../../assets/external/i.imgur.com/e3bb9f7c1731.png)

裝上之後，待機僅有 48 度 ( 當時好像還有跑東西 )  
![](../../assets/external/i.imgur.com/93e7f6bf9fdb.png)

跑個 CPU-Z  
![](../../assets/external/i.imgur.com/df4f06974871.png)

跑個 CINEBENCH ，在超頻情況也只有 70 度，原裝散熱器是跑不完的，會超過溫度上限  
![](../../assets/external/i.imgur.com/67b52dc0cd56.png)

整體至少少了快四成的溫度，而且看起來還綽綽有餘，估計上到 5900 應該都還能撐得住，加上這個價格，真的算是 CP 值非常高的選擇。
